---------------
 MyICQ是什么？
---------------
    MyICQ是一套完全公开源代码的即时通讯软件，包括服务器端和客户端，可以用于互联网或局域网中。可以运行在Windows或Linux(KDE/Qt)操作系统上，这是Windows的客户端。MyICQ完全支持服务器之间的松散连接，这意味着任何人都可以在Internet上架设自己的服务器。群组功能使您能够享受到聊天室和网络游戏所带来的快乐。

---------------
 基本功能特点
---------------

采用MySQL数据库

日志文件

配置文件

NT服务

数据库查询部分采用多线程

系统管理员(MyICQ# = 10000)可以发送广播消息给所有在线和离线的用户

客户端与服务器的通讯采用DES的加密算法

支持服务器松散连接

群组功能，包括聊天室和游戏大厅
